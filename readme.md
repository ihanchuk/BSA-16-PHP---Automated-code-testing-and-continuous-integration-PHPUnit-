# BSA 16 PHP - Task management and job queues in general & in the context of Laravel 5.2

BSA 16 PHP - Automated code-testing and continuous integration (PHPUnit).

## Installation
Clone from GIT:
```sh
$ git clone <this repo>
```

Make a a named host like:
```sh
http://http://dz10.loc
```

Run composer:
```sh
$ composer install
```

Run migrations:
```sh
$ php artisan migrate
```

Run seeder:
```sh
$ php artisan db:seed
```
## Execute test
```sh
$ phpunit
```
